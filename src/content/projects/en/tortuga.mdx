## The challenge

Tortuga's booking conversion rate on mobile was less than half of desktop. Their existing app was a 4-year-old WebView wrapper that opened slow and felt foreign.

## What I built

- Native React Native app with shared design tokens with the web
- Stripe Payment Sheet integration — Apple Pay / Google Pay one-tap checkout
- Offline-first itinerary view using SQLite + sync, so divers on a boat can read tomorrow's plan
- Sentry + custom analytics dashboard for the founders' weekly metrics email

## Results

- Mobile checkout completion went from **31% → 58%** within six weeks
- App Store rating climbed from **3.4 → 4.7** after the rewrite
- Support tickets about "can't book" dropped to near zero

## What I'd do next

Add an onboarding tutorial for first-time divers — most drop-offs now happen at the gear-selection step, not payment.
